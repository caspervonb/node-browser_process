# browser_process.type

## SYNOPSIS

```js
type(command)
```

## PARAMETERS
`command` *String*
:   The command to find the browser type of.

## DESCRIPTION

Returns the type of browser for the given `command`.

Possible values are `chrome`, `chromium` and `firefox`.

## RETURN VALUE

`String`
